# 📅 Booking Slot Selector App

A responsive React-based booking system that allows users to select available time slots for a chosen date. It includes validation rules, slot limits, and a live summary panel with persistent storage.

---

## 🚀 Live Demo
https://booking-slots-phi.vercel.app/

---

## 📌 Features

- 📆 Date selection system  
- ⏰ Time slot selection  
- ⚡ Maximum 3 slots selection limit  
- 🚫 Prevent selecting multiple slots from same hour  
- 💾 Data saved in localStorage  
- 🧾 Live summary panel  
- 🗑️ Remove selected slots  
- 📱 Fully responsive UI (mobile, tablet, desktop)  

---

## 🛠️ Tech Stack

- React.js (Functional Components)
- React Hooks (useState, useEffect)
- Tailwind CSS
- LocalStorage
- Vite / Create React App
